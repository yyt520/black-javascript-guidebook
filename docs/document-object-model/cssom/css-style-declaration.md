# CSSStyleDeclaration

CSSStyleDeclartion 接口表示一个对象，它是一个 CSS 声明块，CSS 属性键值对的集合。它暴露了样式信息和各种与样式相关的方法和属性。

暴露于以下三种 API 中：

- `HTMLElement.style`
- `CSSStyleSheet`
- `window.getComputedStyle`
